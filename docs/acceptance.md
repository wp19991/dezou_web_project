# Acceptance

## Manual Smoke Test

1. 打开首页，创建一个 `2` 或 `6` 人 session。
2. 在创建 Session 时输入固定随机种子，开始连续两手，确认同一 Session 下 hand seed 自动递增派生。
3. 点击“开始新一手”，确认能看到所有底牌、当前行动者与合法动作。
4. 提交若干动作，确认时间线、公共牌与底池同步变化。
5. 发送聊天，确认不影响牌局推进。
6. 结束一手牌后，确认历史列表立即可查询，且无需再开始下一手就能打开 replay。
7. 若进入摊牌，确认赢家、赢取金额、参与比牌座位和最佳牌型都有展示。

## Known Limits

- 仅支持单进程运行时状态。
- 前端采用轮询，不包含 WebSocket。
- 仅提供全明牌调试模式。
